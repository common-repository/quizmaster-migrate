Import quizzes from WP Pro Quiz to QuizMaster.

== Description ==

Import quizzes from WP Pro Quiz to QuizMaster. Requires QuizMaster.

= Features =

* Imports WP Pro Quiz exports in XML format
* Easy to use interface made with ACF Custom Fields

https://github.com/goldhat/quizmaster/wiki

= Support =
Visit https://goldhat.ca/plugins/quizmaster-migrate

== Installation ==

1. Upload the "quizmaster-migrate" folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the new Migrate tab under the QuizMaster menu at QuizMaster > Migrate.

== Basic Usage ==

Export your quiz or quizzes from WP Pro Quiz. Use the XML format (not the custom WPQ format). Paste the quiz XML into the QuizMaster Migrate form and click update. Check the QuizMaster > Questions, and QuizMaster > Quizzes tabs to find your imported items.
